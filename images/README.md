# Pildid / Images

Kõik referentspildid on kategooriate kaupa kaustades. Lehe galerii viitab nendele
teedele, seega hoia struktuur samasugune (uusi võib alati juurde lisada).

```
images/
├── exterior/                 # Väline kujundus (peeglid, kalasoomused)
│   ├── scales-closeup.jpg     – pärlmutterläikega soomuste lähivõte
│   ├── roof-peak.jpg          – teravatipuline katusenurk
│   └── full-structure.jpg     – klaaspaviljon, soomuseline serv
└── interior/                 # Sisemus (veealune maailm)
    ├── walls/                 # Seinad – walls-1.jpg … walls-7.jpg
    ├── fish/                  # Kala kujutis – fish-1.jpg … fish-5.jpg
    ├── ceiling/               # Lagi ja kangad – ceiling-1.jpg … ceiling-4.jpg
    └── seating/               # Keskmine istumisala – seating-1.jpg … seating-5.jpg
```

## Piltide lisamine / Adding images

- Lae alla URL-ist: anna mulle link, ma laen alla ja panen õigesse kausta.
- Või lisa fail ise õigesse kausta ja järgi nimetamismustrit (`kategooria-N.jpg`).
- Soovitus: hoia pildid ~1600px pikemalt küljelt, JPEG ~80 — kiire laadimine.
